# Automated DCF Valuation Platform

## Overview

A Python-based DCF valuation platform that automates:

- Financial statement extraction
- FCFF forecasting
- WACC calculation
- CAPM valuation
- Sensitivity analysis
- Investment recommendations
- Excel report generation

## Features

- Global stock support
- Automated revenue forecasting
- Automated tax rate estimation
- Automated depreciation and CapEx assumptions
- Automated WACC calculation
- Sensitivity analysis
- Multi-sheet Excel reports

## Technologies

- Python
- Pandas
- OpenPyXL
- Yahoo Finance API
